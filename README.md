# FiveM Blip Builder

A simple web tool for FiveM devs: generate a clean **Lua blip snippet** from basic inputs.

## Features
- Set coords (X/Y/Z)
- Sprite, color, scale, name
- Short range toggle + display mode
- Optional: category + route + route color
- One-click copy of the full Lua snippet

## Usage
1. Download / clone this repo
2. Open `index.html` in your browser

## Demo
Click **Demo** in the UI to generate an example blip.
